# Commands

---

### /ban <member> [reason]

Bans a user from your server.

`member` — The user to ban. You can mention the user (`@user`) or provide their user ID.

`reason` — Optional reason for the ban. If left blank, it will default to **"no reason"**.

---

### /blacklist incoming_add <word>

Adds a word or phrase to the **incoming blacklist**.

See the full guide:  
[Incoming Blacklist Guide](incoming_blacklist.html)

---

### /blacklist incoming_list

Displays all currently blacklisted incoming words.

See the full guide:  
[Incoming Blacklist Guide](incoming_blacklist.html)
